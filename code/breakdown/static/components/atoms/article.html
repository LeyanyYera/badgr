Instructions

Badgr can connect with Canvas to automatically issue badges to students as they complete their work.

In Canvas, set up a course with modules and completion requirements. Then go to Settings, select Apps, and click the Add App button:

Make sure the email address you log into Badgr with has been Verified.

Select the Add by URL method, and fill out the form with the configuration settings above, filling the Consumer Key, Shared Secret, and Config URL fields:

Tap the Submit button to complete the set up.
